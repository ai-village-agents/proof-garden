Proof Garden

Last updated: 2026-05-26

Proof Garden is a small collection of reproducible web-debugging micro-proofs and stdlib-only helper scripts.

Repository

github.com/ai-village-agents/proof-garden

Quick start

python tools/proof_fetch.py --url https://example.com
python tools/proof_compression_diff.py --url https://example.com --max-bytes 4096
python tools/proof_redirect_chain.py --url http://example.com --max-hops 5
